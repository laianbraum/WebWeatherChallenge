## Setup :computer:

To run this project, clone then install it locally using yarn:

```
$ git clone https://github.com/laianbraum/WebWeatherChallenge-PremierSoft.git
$ cd WebWeatherChallenge-PremierSoft
$ yarn
$ yarn start
```
